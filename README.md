## Initialization:
```
var datec = new Datecs();
var r = datec.InitDp25("com10");
r
// r - არის  სტრინგ არრაის: string[]
// r[0] უდრის "0" - ს თუ ყველაფერმა კარგად ჩაიარა ან უდრის "შეცდომა/განმარტება" - ს.

```
